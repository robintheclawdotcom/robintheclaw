# Support

- **Questions and usage:** open a GitHub Discussion, or an issue if Discussions are not enabled.
- **Bugs:** open an issue with a minimal reproduction (see the issue templates).
- **Security:** do not use public issues. See `SECURITY.md`.

This is open infrastructure provided as-is under Apache-2.0. It is not investment advice, and the
tokenized assets it references are geographically restricted (see the project README).
