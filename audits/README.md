# Audits

Security reviews of the Robin the Claw contracts and verifier are collected here.

No third-party audit has been completed yet. The contracts are pre-audit. Unaudited code should not
custody funds it cannot afford to lose.

The current audit handoff is [scope-v1.md](scope-v1.md). It defines the intended dormant mainnet
deployment profile and the invariants an independent review must assess.

When a review is completed, the report is added here along with its exact scope: the commit hash
reviewed, the contracts covered, and the resolution of each finding. Releases reference the audit
that applies to them.
